Для запуска проекта:
1) Скачать проект
2) Перейти в корневую папку
3) В консоли прописать ``` npm install``` (Для этого на компьютере надо иметь nodeJs и npm)
4) После того, как все модули скачаются запускаем проект с помощью команды ```npm start```
